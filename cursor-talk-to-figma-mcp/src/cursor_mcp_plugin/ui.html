About Token Link

Token Link helps you keep Figma and your design tokens in sync. It reads variables from selected elements and components, matches them to your design tokens, and smartly applies the right variables back into Figma â€” so components become tokenized without manual plumbing. It is based on the original cursor-talk-to-figma-mcp project and has been customized for token-driven workflows, reverse tokenization, and tighter integration with our design system.

Version: 1.0.0 (custom fork)

How to Use

  1. Make sure the MCP server is running in Cursor
  2. Connect to the server using the port number (default: 3055)
  3. Once connected, you can interact with Figma through Cursor
